# Fake News Blocker

A machine learning project to classify news as Fake or Real using TF-IDF + Logistic Regression.

Includes:
- Training script (main.py)
- Flask deployment (app.py)
- Test script (test_fake_news.py)
- Fake.csv & True.csv dataset
- Saved model (fake_news_model.pkl)
- TF-IDF vectorizer (vectorizer.pkl)

Run Flask app:
python app.py
